# ⚡ tes - Modern Premium E-Commerce Experience

tes is a high-end Django-based e-commerce platform designed with a focus on premium aesthetics, seamless user experience, and modern web standards. Built using **Django 6.0**, **Tailwind CSS**, and **DaisyUI**, tes offers a sleek, responsive interface that feels alive and interactive.

![tes Brand Identity Header](https://raw.githubusercontent.com/TNlucfer01/Django_mps/Testing/theme/static/images/hero.png)

## 💎 Key Features

- **Premium UI/UX**: Replicated luxury retail aesthetics with vibrant purple-blue gradients, glassmorphism, and modern typography.
- **Dynamic Content**: Responsive home page featuring curved hero sections, assurance grids, and sleek product displays.
- **Full Shopping Suite**: Complete flow including Product Discovery, Cart Management, and Checkout.
- **Order Tracking**: Comprehensive order list and detail views with colorful status indicators.
- **Modern Admin Panel**: Enhanced Django Admin interface using **Jazzmin** for a professional management experience.
- **Authentication**: Fully styled login, registration, and user profile management pages.
- **Mobile First**: Fully responsive design that ensures a premium experience across all devices.

## 🛠️ Tech Stack

- **Backend**: Python 3.13, Django 6.0.3
- **Frontend**: Tailwind CSS v4, DaisyUI 5, Vanilla JavaScript
- **Admin**: Django-Jazzmin
- **Database**: SQLite (Development) / PostgreSQL (Production ready)
- **Tooling**: Django-Tailwind integration for seamless CSS builds

```text

├── products/              # Product catalog & details
├── cart/                  # Shopping cart logic
├── orders/                # Order processing & history

└── static/                # Global static assets
```
